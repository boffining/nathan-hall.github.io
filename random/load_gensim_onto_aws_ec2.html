Load Gensim onto AWS EC2

  1. 02 April 2017
  2. random

Gensim is truly awesome. However, getting it loaded on an EC2 instance with the right C compiler can be tricky. Especially since Gensim will continue to work with its "slower version" which is expensively time consuming.

The fast version is 70x faster than the slow version.

Load the dev tools onto the instance

Run the below command to install the dev tools that includes the gcc compiler needed for gensim.

$ sudo yum groupinstall "Development Tools"

Do the easy_install of gensim instead of pip.

Typically this isn't done, but for some reason the easy_install version sets all the correct paths to the C compiler so that the fast version can be used.

$ easy_install --upgrade gensim

Check that fast version is working

# To check if you have the fast version installed you do the following.
from gensim.models.word2vec import FAST_VERSION
# if you get a 1 you have the fast version, a -1 means you have the slow version.
FAST_VERSION
