Dictionary Comprehension for List of Lists

  1. 26 November 2016
  2. python

Turn a list of lists into a dictionary with one line of code... the magical comprehension.

Click here for a refresher on how to read in data with the csv.reader

import csv

with open('../data/sales_data.csv', 'rU') as f:
    data = []
    reader = csv.reader(f)
    for row in reader:
        data.append(row)
    print data[0:5]

[['volume_sold', '2015_margin', '2015_q1_sales', '2016_q1_sales'], ['18.4207604861', '93.8022814583', '337166.53', '337804.05'], ['4.77650991918', '21.0824246877', '22351.86', '21736.63'], ['16.6024006077', '93.6124943024', '277764.46', '306942.27'], ['4.29611149826', '16.8247038328', '16805.11', '9307.75']]

Use the first list item (column headers) as the dictionary keys.

# Getting the header.
header = data[0]

# Drop the first list item with 'pop'.
data.pop(0)

['volume_sold', '2015_margin', '2015_q1_sales', '2016_q1_sales']

Now for that scary comprehension thingy.

"""This is using the column name as the key and then a list comprehension to embed all the
data that belongs that column as a list for the value.

The enumerate is returning the index of the column header so that we can get the data
from each subsequent list item at that index location.

Also, convert the string data from the csv into float.
"""

sales_data = {column_name: [float(row[i]) for row in data] for i, column_name in enumerate(header)}

# Now easily get the sum of volume sold.
sum(sales_data.get('volume_sold'))

2003.73681579776
