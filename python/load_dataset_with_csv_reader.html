Load Dataset with CSV.reader

  1. 23 November 2016
  2. python
# Import the csv module that is built in to python.
import csv

# Open the file and assign its reference to the variable "f". The "rU" is the universal newline way to open a csv.
with open('../data/sales_data.csv', 'rU') as f:
    data = []
    reader = csv.reader(f)
    for row in reader:
        data.append(row)
    print data[0:5]  # print the column headers as the first line item and then each row of data in the next line.
#f.close() -- each with statement has a close statement built into the block so this is not needed.
